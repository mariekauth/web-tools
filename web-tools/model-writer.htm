Model Writer

This tool will write a model using the PSR standards with the proper spacing and comments to be picked up by more robust IDE (Intergrated Development Enviroment) software editors. Since I am writing the code using a basic text editor, but want it to already be properly formatted and work when edited with an IDE I thought this tool would be useful.

Of course if you are using the symfony framework, and have a database that is already properly written using Doctrine and ORM, then this tool is useless.
