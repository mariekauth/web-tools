On March 6, 2017 I lost my job and find myself without an up to date web portfolio. Over the past 6 months I have struggled greatly because I was learning a lot of new skills at once and did not have the opportunity to digest all that I had learned.

With a lot of new skillsets and seeing things from a different view I am taking stock of what I know, and creating a new portfolio that will demonstrate, at least to the most basic level, my understanding of each of thos skills.

Objectives

  • Quickly develop web tools using MVP (Mininum Viable Product)
  • Demonstrate the ability to write Object Oriented code in multiple programming languages.
  • Create tools that will help me write code in multiple languages with the minimal amount of effort.
  • Demonstrate the ability to use Git and GitHub.
  • Allow future employers to be able to see how I work and how I think

Please keep in mind that at this time this is only a DRAFT. Normally, you would not post your draft to the Web EVER. I am doing so for very specific reasons.

My ultimate goal here is to get a job. To properly update my resume I need to take stock of the skills I currently have and what level they are really at. I also would like to have a tool box that I can take with me from company to company for when the company I am working for does not have those tools already available.

Programming is an area where only the end product is available. Perspective employers have no way of knowing how a candidate thinks, how long a project takes for a candidate to complete, by what method they came up with the completed project.

My thought is that I can hone my git and scrum skills, access my current levels in each area, demonstrate my skills and abilities, show perspective employers how I think and how I work, get my resume updated, develop my toolbox, and find a job all at the same time.

Posting at GitHub, and posting many of my 'Drafts' as live allows me to do all of the above. Once this particular project is completed drafts will no longer be posted to this site, but only finished projects.
